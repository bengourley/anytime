Anytime

A JS date and time picker

  • Say goodbye to jQuery UI!
  • Modularity FTW!
  • Only 4 lines of essential CSS (the rest is up to you)
  • Use with browserify, AMD or standalone
$ npm i --save anytime

Usage

Browserify

This is the recommended and best supported way to use Anytime. Install anytime with npm and require it like so:

var Picker = require('anytime')

AMD

If you’re an AMD user, luckily browserify makes it easy for me to export a UMD version of the module. Go grab the anytime.standalone.js file from the repo.

<script src="anytime.standalone.js"></script>

Browser Global

Yes it’s evil but I’ll let you do it. Here's how to use anytime as a global…

<script src="anytime.standalone.js"></script>
<script>
  var Picker = window.anytime
</script>
